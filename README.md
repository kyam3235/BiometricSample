# BiometricSample

## 環境
- Android Studio 3.6.3

## 概要

[Biometric](https://developer.android.com/jetpack/androidx/releases/biometric?hl=ja)を使ったサンプルアプリ

## 使い方

1.[生体認証]ボタンをタップすると指紋認証のダイアログが表示される。  
2.指紋認証の認証成功/失敗はLogcatに出力される。
